## **README.md tipos de pericias e sua Aplicação. Imersão DEV com Google Gemini ** 

###  **O que faz essa aplicação?**

Essa aplicação é como um detetive particular digital! ️‍♀️ Ela te ajuda a encontrar as informações que você precisa sobre diferentes tipos de perícias, como grafoscopia, papiloscopia e crimes digitais. É só digitar uma palavra-chave e pronto! A mágica acontece. ✨

### ️ **Como funciona a mágica?**

1. **Você digita:** Escreve o tipo de perícia ou uma palavra-chave no campo de pesquisa.
2. **A aplicação busca:** Vasculha uma base de dados cheia de informações sobre diferentes tipos de perícias.
3. **Resultados na tela:** A aplicação te mostra os resultados que combinam com a sua pesquisa, de forma clara e organizada.

###  **Tecnologias utilizadas:**

* **HTML:** A estrutura da página, como se fosse a ossatura de um site.
* **CSS:** A parte que deixa tudo bonito e organizado, como um bom designer de interiores.
* **JavaScript:** O cérebro da aplicação, responsável por fazer tudo funcionar e responder às suas interações.

###  **Estrutura dos arquivos:**

* **index.html:** O arquivo principal, que você abre no navegador.
* **styles.css:** Contém as regras de estilo para deixar a página bonita.
* **dados.js:** Armazena as informações sobre as perícias (a base de dados).
* **app.js:** O arquivo JavaScript que faz toda a lógica da pesquisa.

###  **Como usar:**

1. **Clone o repositório:** Baixe o código para o seu computador.
2. **Abra o index.html:** Dê um duplo clique no arquivo para abrir a página no seu navegador.
3. **Faça sua pesquisa:** Digite o que você procura e clique em "Pesquisar".

###  **Dicas:**

* **Seja específico:** Quanto mais precisa for a sua pesquisa, melhores serão os resultados.
* **Explore as opções:** Clique nos links para saber mais sobre cada perícia.
* **Dê um feedback:** Se você encontrar algum problema ou tiver alguma sugestão, me avise!
